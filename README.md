# Reactivities.NET

## Installation

1. Install .NET 9 and run from the API folder:
   ```bash
   dotnet restore
   dotnet run
   ```
2. From the client folder:
   ```bash
   npm install
   npm run dev
   ```

## Usage

- Runs on localhost with .NET backend and React frontend
- Provides Activities CRUD operations
